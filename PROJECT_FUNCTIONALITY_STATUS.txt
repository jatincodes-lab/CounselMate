CounselMate CRM - Functionality Status
Updated: 2026-07-07

Project Summary
CounselMate CRM is a multi-tenant SaaS CRM for coaching institutes and admission teams. It supports institute onboarding, branch and team management, lead management, follow-ups, admission pipeline tracking, documents, payments, applications, enrollments, notifications, reports, and role-based access.

Technology Covered
- Frontend: React + Vite CRM interface.
- Backend: ASP.NET Core Web API.
- Database: PostgreSQL with Entity Framework Core and Npgsql.
- Background jobs: Hangfire foundation for reminders and automation.
- Deployment notes: Neon PostgreSQL, Render backend, and Vercel frontend.

Known Notes
- PowerShell blocks direct npm script execution on this machine, so use npm.cmd for frontend commands.
- Archived leads are read-only until restored.
- Follow-up actions use optimistic version checks.
- Lead writes use optimistic version checks.
- Completed and cancelled follow-ups are immutable.
- CRM timestamps are standardized to Indian Standard Time for persisted business data.
